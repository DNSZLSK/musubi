# MUSUBI

A retro CRT-style logic puzzle game built with vanilla JavaScript.

**[Play Now](https://dnszlsk.github.io/musubi/)**

![MUSUBI Screenshot](https://raw.githubusercontent.com/DNSZLSK/musubi/master/public/Images/musubi%20logo%204.png)

## How to Play

- Tap circles to fill them
- Each number shows how many neighbors are filled
- Match all numbers to win!

## Install on your phone

### iPhone (Safari)
1. Open the game in Safari
2. Tap the Share button (square with arrow)
3. Tap "Add to Home Screen"
4. Tap "Add"

### Android (Chrome)
1. Open the game in Chrome
2. Tap the menu (three dots)
3. Tap "Install app" or "Add to Home Screen"

## Tech Stack

- **Vite** - Build tool
- **ESLint + Prettier** - Code quality
- **Vitest** - Testing
- **GitHub Actions** - CI/CD
- **PWA** - Installable on mobile

## Development
```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm run test

# Build for production
npm run build
```
